On this page you can set encryption key which LemonStand uses for encrypting and decrypting data in the database. Important! If there is any encrypted data in the database, changing the encryption key will lead to impossibility to decrypt the data. You may need to change the encryption key if you are creating a new database or restoring the database from LemonStand archive. You need to know the encryption key for each database you use. It is not necessary to change the encryption key for each new database.

After changing the encryption key you may be automatically logged out of the Configuration Tool.
